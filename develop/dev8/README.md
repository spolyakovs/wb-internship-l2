## Task 8 - Unix-shell

Imitation of Unix-shell
## Usage:
```
$ go run ./
```
Then type commands

### Commands:

```
cd <dir>
```
change directory to `dir`

```
pwd
```
print current working directory

```
echo <args...>
```
print `args`, separated by space

```
kill <pids...>
```
send kill signal to processes with PIDs=`pids`

```
ps
```
print all current processes
